# 📖 Reciter - 高考语文背诵辅助工具

> *Note: Written by 哈基米 3 Pro*

这是一个专为高考古诗文默写和记忆设计的 Web 应用程序。它提供智能化的挖空背诵、上下句测试以及可视化的重点标记功能，所有数据支持本地持久化存储。

## ✨ 主要功能

### 1. 智能背诵模式
*   **多种遮盖策略**：支持“全遮盖”、“提示首字”、“随机按比例挖空”等。
*   **智能分词**：自动识别汉字与标点，挖空时保留标点符号，维持句读节奏。
*   **悬停提示**：鼠标悬停（或点击）在遮盖的方框上，可临时偷看答案。

### 2. 交互式学习与测试
*   **⚡️ 上下句测试**：随机抽取文章中的一句，要求补全上句或下句，强化记忆连接。
*   **可视化标记**：支持在正文中划词，添加**红字**、**绿字**、**下划线**、**高亮**等标记，用于强调易错字或重点实词。也支持键盘快捷键操作，请自行探索。
*   **实时保存**：所有标记操作会自动同步保存到本地 JSON 文件，刷新页面不丢失。

### 3. 现代化 UI 与体验
*   **深色模式**：一键切换 Light/Dark 主题，舒适护眼。
*   **打印支持**：完美适配 A4 纸张的打印样式，可直接导出为纯净的 PDF 默写纸。
*   **字体优化**：采用与人教版教材一致的方正书宋字体，确保视觉效果高度统一。

## 🛠 技术栈

*   **Frontend**: 原生 HTML5, CSS3 (CSS Variables, Flex/Grid), Vanilla JavaScript (ES6+)。
*   **Backend**: Python 3 (用于搭建轻量级 API Server，处理 JSON 数据的读写)。
*   **Data**: JSON (存储文章内容及标记掩码)。

## 🚀 快速开始

### 1. 启动服务
在项目根目录下打开终端（Terminal / CMD），运行：

```bash
python server.py
```

### 2. 访问应用
打开浏览器，访问：
`http://localhost:1075/`

## 📂 目录结构

*   `index.html`: 前端主入口。
*   `server.py`: Python 后端脚本，提供静态服务及 `/save` 接口。
*   `data/`: 存放文章数据 (`.json`) 和目录索引 (`catalog.json`)。

## 📝 如何添加新文章

1.  在 `data/` 目录下新建 JSON 文件（如 `new_article.json`）。
2.  仿照现有格式填写 `id`, `title`, `content` 等字段。
3.  在 `data/catalog.json` 中注册该文件即可。
